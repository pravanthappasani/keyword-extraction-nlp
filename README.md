# Keyword Extraction using NLP

## Overview

This project demonstrates keyword extraction using Natural Language Processing (NLP) and TF-IDF. It preprocesses text and extracts important keywords from scientific documents.

## Features

- Text preprocessing
- Tokenization
- Stopword Removal
- Lemmatization
- TF-IDF Keyword Extraction

## Technologies

- Python
- NLTK
- Scikit-learn
- Pandas
- NumPy

## Future Improvements

- SRP-TF-IDF
- BERT Integration
- Web Interface
